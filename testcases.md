# #1 Рыночная покупка акций компании
### Предусловие
Открыта главная страница сайта trading.finam.ru
Есть средства на покупку акций

| Шаг | Ожидаемый результат |
| --- | ------------------- |
| В верхнем меню сайта нажать на кнопку "Заявка" | Открылось окошко с заявками c открытой вкладкой "Купить" и подраздрелом "Рыночная" |
| Ввести в поле "Количество" необходимое количество акций для покупки | В поле отображается введенная информация | 
| Сверить, что конечная цена покупки корректно отображается в кнопке "Купить" | Цена динамически изменяется в зависимости от стоимости и желаемого количества акций |
| Нажать на кнопку "Купить" | Открылось окно с контрольной информацией о заявке |
| Нажать на кнопку "Подтвердить" | Появилась информация о подтверждении выполнения заявки |
| Проверить наличие купленных акций в разделе "Портфель" | В разделе "Портфель" отображаются купленные акции |

# #5 Создание алерта
### Предусловие
Открыта страница сайта trading.finam.ru

| Шаг | Ожидаемый результат |
| --- | ------------------- |
| В верхнем меню сайта нажать на значок колокольчика с плюсом | Открылось окошко создания алерта |
| В левом меню выбрать условие срабатывания алерта | Выбранное условие отображается в поле левого меню |
| В правом меню выбрать значение, при котором условие сработает | Выбранное значение отображается в поле правого меню|
| В самом нижнем меню выбрать способ срабатывания алерта | Выбранный пункт отображается в поле нижнего меню |
| Нажать на кнопку создать алерт | Окошко создание алерта закрылось |
| Проверить наличие созданного алерта в разделе "Алерты" в левой стороне страницы | В разделе "Алерты" отображается созданный алерт |
| Проверить оповещение во вкладке "Уведомления" срабатывание алерта при достижения поставленных условий | Проигрывается звук и во вкладке "Уведомления в левой стороне страницы появляется пункт срабатывания алерта |
